# CodeCraft API

API básica en Node.js para gestión de tareas.

## Endpoints
- `GET /tasks`: Lista todas las tareas.
- `GET /tasks/:id`: Muestra una tarea por ID.

## Comandos útiles

```bash
npm install
npm start
npm test
